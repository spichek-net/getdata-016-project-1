# Data dictionary
## Subject
A number uniquely identifying test subject
Type: Positive integer
## Activity
Name of activity the subject was performing when measuremnents were taken
Type: Character string
Allowed values: 
.WALKING
.WALKING_UPSTAIRS
.WALKING_DOWNSTAIRS
.SITTING
.STANDING
.LAYING
